# store.latexdo.org

This repository is reserved for the future LatexDo store site at `https://store.latexdo.org`.

## Repository Role

- Placeholder for future store, licensing, checkout, or account-related pages.
- Currently contains no runnable application code.
- Should follow the same README structure as the other LatexDo web projects when implementation starts.

## Requirements

There are no runtime requirements yet.

## Run Locally

There is nothing to run locally yet. When the store app is added, keep the local workflow in this shape:

```sh
npm install
npm run build
npm run dev
```
